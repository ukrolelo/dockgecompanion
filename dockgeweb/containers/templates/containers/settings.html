About Dockge Companion

Dockge Companion is a Docker container digest monitoring tool that helps you track changes to your container images over time. It provides both command-line and web interfaces for managing your container infrastructure.

Key Features:
  • Track container digest changes automatically
  • Check for available image updates
  • View detailed container history
  • Generate comprehensive reports
  • Get rollback instructions for previous versions
  • Web and command-line interfaces
Command Line Usage:
# Initialize tracking
python dockge_companion.py init

# Scan containers
python dockge_companion.py scan

# Check for updates
python dockge_companion.py check-updates

# View status
python dockge_companion.py status

# Start TUI
python dockge_companion_tui.py
Tips
  • Run regular scans to track changes
  • Check for updates periodically
  • Use the auto-refresh feature on status pages
  • Copy digest values for rollback operations
  • Monitor the recent changes for activity
Important

Always backup your docker-compose.yml files before performing rollback operations. Test rollbacks in development environments first.
